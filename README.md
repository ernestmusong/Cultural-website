### Preriquisites

Ruby version >= 3.0.0
Node
PostgresQL

### Setup

Clone this repository to your desired folder:
sh
  cd my-folder
 git clone https://github.com/ernestmusong/Budget-app.git

 ### Install:
 Run: bundle install
 Run: npm install

 ### Create database:
 Run: rails db:create

 ### Create database schema:
